# Countries-gdp-growth-prediction-and-arima-forcast-
functions that display growth trends for any country in the worldbank dataset, predict gdp 10 years from 2022 based on historic data using linear regression and displays an arima forcast for 10 years with moving average.

the ipynb file contains the three files and csv file contains the data. keep both in same directory.

you can put any name of any country into all of the functions and any date up from 1960 to 2022. the countries must have capital first letters and some are abbreviated such as "USA".

functions automattically clean the data and some more obsure countries might not be avalible since they have so much missing data.

second ipunb file contains a function to display the percentage of a countries economy that derives from internal consumption. keep all files in the same directory.
